## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```sh
$ npm install
$ cp .env.example .env
# BD
$ docker-compose up -d
```

## Running the app

```sh
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
